سياسة الخصوصية

نحن نقدر خصوصيتك ونلتزم بحماية بياناتك أثناء استخدام التطبيق.

📱 وصف التطبيق

التطبيق هو مشغل صوت متقدم يتيح لك إنشاء وإدارة قوائم تشغيل غير محدودة مع تجربة تشغيل سلسة وحديثة تشبه التطبيقات الاحترافية.

يمكنك إضافة ملفات صوتية بسهولة، تشغيلها مباشرة داخل التطبيق، والتحكم الكامل في الصوت مثل الإيقاف، الإيقاف المؤقت، والتنقل بين المقاطع.

يدعم التطبيق حفظ القوائم تلقائيًا، وتشغيل الصوت في الخلفية، مع واجهة حديثة تعتمد على تصميم أنيق وشفاف (Glass UI).

يتميز أيضًا بإمكانية حذف أو تعديل القوائم بسلاسة عبر السحب (Swipe Actions)، مما يجعل التجربة أسرع وأكثر احترافية.

⭐ المميزات

  • إنشاء عدد غير محدود من قوائم التشغيل
  • إضافة ملفات صوتية بسهولة باستخدام File Importer
  • تشغيل / إيقاف / إيقاف مؤقت للصوت
  • تشغيل المقطع التالي تلقائيًا
  • حفظ تلقائي للقوائم باستخدام UserDefaults
  • إدارة الملفات داخل كل قائمة
  • حذف القوائم أو الملفات عبر السحب (Swipe Actions)
  • إعادة تسمية القوائم
  • التحكم في مستوى الصوت
  • شريط تقدم الصوت (Slider)
  • متابعة وقت التشغيل الحالي

🔒 الخصوصية

التطبيق لا يجمع أي بيانات شخصية من المستخدمين. جميع الملفات يتم حفظها محليًا داخل الجهاز فقط ولا يتم إرسالها لأي سيرفر خارجي.
